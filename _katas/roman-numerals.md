Given an integer as input, we want to produce the [Roman Numeral](https://en.wikipedia.org/wiki/Roman_numerals) equivalent output as below:

```ruby
roman_numeral(2)
=> "II"

roman_numeral(4)
=> "IV"

roman_numeral(5)
=> "V"

roman_numeral(10)
=> "X"
